I meet a problem, when I want to copy a 4.8G file to a friend.
Why not AirDrop? I really want to use, but his computer is just snow lepoard.(I use Lion)

So I decied to split this file into three 2G file
which means each file is 2000m

[sourcecode language="bash"]
$ split -f 2000m filname newFileName

$ cat newFileNameaa newFileNameab newFileNameac &gt;filename
[/sourcecode]

The problem I meet is when you split a large file, it eat up all your inactive memory?
